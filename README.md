# Ilgın Akkoyun - Engineering Portfolio

Welcome to my personal engineering portfolio! This repository contains the source code for my interactive CV website.

## 🚀 About This Project

This responsive web application showcases my academic background, professional experience, and engineering projects in the field of Electrical Engineering and Energy Systems. It is designed to be fully bilingual (Turkish/English) and mobile-friendly.

### Key Features:
*   **Dual Language:** Seamlessly switch between Turkish and English.
*   **Interactive Project Modals:** Detailed views of my engineering assignments (Power Electronics, Circuit Analysis, Lighting Design, BESS Arbitrage) complete with simulation data and parameters.
*   **Dynamic HUD:** Clean, modern interface built with Tailwind CSS.
*   **Fully Responsive:** Optimized for both desktop and mobile viewing.

## 🛠️ Technologies Used
*   **HTML5 & CSS3**
*   **JavaScript (Vanilla)**
*   **Tailwind CSS** (for styling and responsiveness)
*   **Lucide Icons** (for UI iconography)
